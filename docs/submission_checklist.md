# Submission Checklist

## Before Recording

- Run `pytest --cov=src --cov-report=term-missing`.
- Run `ruff check .`.
- Run `mypy src tests`.
- Run `python -m src.main`, then test `build`, `load`, `print`, and `find`.
- Confirm `data/index.json` has been generated by the real `build` command.
- Check the Git log shows regular, meaningful commits.

## Video Must Show

- `build` crawling and saving the index.
- `load` loading the saved index.
- `print <word>` showing URL, frequency, and positions.
- `find <single word>` returning ranked pages.
- `find <multi word query>` returning ranked pages.
- Edge cases: missing word and empty query.
- Tests running successfully.
- Git commit history.
- GenAI declaration and critical evaluation.

## Minerva Submission

Submit one PDF or TXT containing:

- Video link.
- GitHub repository URL.
- Index file attachment or download link.

Before submission, open the video and repository links in an incognito/private
browser window to confirm marker access.
